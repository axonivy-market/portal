ポータルのコンポーネント
**********************************************

``portal-components`` に、ポータルに依存しないコンポーネントを導入しました。
これらを使用する場合、ポータルのプロジェクトをインポートする必要はありません。

これらのコンポーネントは、PrimeFaces の Freya テーマと Serenity テーマの両方を完全にサポートします。
