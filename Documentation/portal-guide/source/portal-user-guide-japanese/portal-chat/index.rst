ポータルチャット
**********************************

ポータルは、ケースに関連するプロセスチャットや 2 人のユーザー間のプライベートチャットをサポートします。チャットは自動的に非アクティブ化されます。


ハウツー：チャットをアクティブにする
---------------------------------------------------------------------

:guilabel:`Portal.Chat.EnablePrivate` ポータル設定でプライベートチャットをオンにできます。

:guilabel:`Portal.Chat.EnableGroup` ポータル設定でプロセスチャットをオンにできます。

:ref:`ポータルの設定の更新 <update-portal-settings-ja>` を参照してください。


ハウツー：ケースに関連するプロセスチャットを作成する
-------------------------------------------------------------------------------------

.. note::

   ケースのプロセスチャットにロールを事前に定義して、そのロールを持つすべてのユーザーをプロセスチャットに自動的に参加させることができます。 

#. 任意のタスクに移動します。プロセスチャットのアイコンが表示されます。

   |create-new-process-chat|

#. ログインユーザーが事前に定義されたロールに属する場合、:guilabel:`ユーザーをチャットに追加` ダイアログが表示されます。プロセスチャットのメンバーを選択できます。
   メンバーはユーザーまたはロールで選択できます。選択ごとに :guilabel:`追加` をクリックします。メンバーを追加した後、:guilabel:`プロセスチャットの作成` をクリックします。

   |chat-process-assignee|

   ログインユーザーが事前に定義されたロールに属さない場合、プロセスチャットが作成されると通知されます。

ハウツー：チャットメッセージを表示／送信する
---------------------------------------------------------------------

#. 右上隅にある |bubble-icon| アイコンをクリックします。

   |access-chat|

#. プロセスチャットまたはユーザーをクリックして、メッセージを表示します。

   |chat|

#. メッセージを入力し、:kbd:`Enter` を押すか、|send-email-icon| をクリックしてメッセージを送信します。
