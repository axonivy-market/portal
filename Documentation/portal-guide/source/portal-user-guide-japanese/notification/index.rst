通知
**************

ポータルの通知には、通知設定に基づいてすべての通知が表示されます。 

|notification-panel|
   #. 通知を開くには、右上隅にあるベルのアイコンをクリックします。すべての通知を含むパネルが表示されます。
   #. パネルでは、:ref:`通知設定 <notification-channels-ja>` にアクセスできます。また、パネルの右上にある［その他のアクション］アイコンをクリックして、全通知ページにアクセスできます。
   #. |show-more-icon| アイコンをクリックして、全通知ページに移動します。

.. note::

   ウェブ通知チャネルが管理者またはユーザーによって無効化されている場合、ベルのアイコンは自動的に非表示になります。
