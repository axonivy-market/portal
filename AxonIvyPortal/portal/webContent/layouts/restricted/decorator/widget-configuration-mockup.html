📌 Layout Design Notes

  • Checkboxes trên, Dropdown dưới: Nhóm cùng loại control lại gần nhau — giúp user scan nhanh.
  • Separator line giữa nhóm checkbox và dropdown → tạo visual grouping rõ ràng.
  • Label bên trái, control bên phải (justify-content: space-between) → pattern chuẩn cho settings panel.
  • Hover highlight mỗi row → cải thiện tương tác.
  • Info icon (ℹ️) chỉ hiện cho option cần giải thích — không spam.
  • Ellipsis cho label dài (text-overflow: ellipsis) → đảm bảo panel không bị vỡ layout.
  • Width cố định (~380px) → phù hợp overlay panel, không quá rộng, không quá hẹp.
  • Variant B phù hợp khi có nhiều option (6+), cần phân nhóm rõ ràng.
  • Variant C phù hợp khi overlay cần gọn nhẹ, ít option.
