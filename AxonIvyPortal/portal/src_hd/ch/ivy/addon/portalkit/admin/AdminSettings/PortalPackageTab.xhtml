Export

Export the current Portal configuration as a zip package. The following files will be included:

  • Portal_Dashboard.json
  • Portal_CustomStatistic.json
  • Portal_UserMenu.json
  • Portal_CaseDetails.json
  • Portal_ThirdPartyApplications.json
Only non-empty configurations are included in the package.

Import

Upload a Portal_Package.zip file. Existing data for each type will be replaced.

Select a Portal_Package.zip file to upload. All configurations included in the package will replace the existing ones.
