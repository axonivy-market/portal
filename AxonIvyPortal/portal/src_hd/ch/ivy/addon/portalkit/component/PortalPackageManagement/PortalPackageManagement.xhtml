Manage Portal configuration files as a portable zip package. Use Export to download all current JSON configuration files bundled into a single zip archive, or use Import to upload a zip package and apply its configurations — replacing the existing ones.

Export

Export the current Portal JSON files as a zip package. The following files will be included:

File Name Description
Portal_Dashboard.json Dashboard widget layouts and configurations for all Portal dashboards
Portal_CustomStatistic.json Custom statistic chart definitions displayed on Portal dashboards
Portal_UserMenu.json User menu items and shortcuts shown in the top navigation bar
Portal_CaseDetails.json Case detail page layouts including visible fields and widget arrangements
Portal_ThirdPartyApplications.json Third-party application links integrated into the Portal menu
Portal_CustomMenuItems.json Custom ordering of items in the left-side navigation menu
Portal_ExternalLinks.json External URL links displayed in the Portal navigation
Portal_MenuOrder.json Custom ordering of the main navigation menu items in the Portal sidebar
Only non-empty files are included in the package.
