XSS Demo: JavaScript Injection via PortalNavigatorInFrameAPI
This demo shows how passing user-controlled input to navigateToUrl() can lead to XSS.

How it works

The navigateToUrl() method URL-decodes the input and concatenates it into a JavaScript string executed by PrimeFaces. A single quote in the decoded URL breaks the JS string context.
